# PyTorch REINFORCE

<img src="assets/algo.png" width="800">

PyTorch implementation of REINFORCE.
This repo supports **continuous** environments in OpenAI gym.


## Requirement
- python 2.7
- PyTorch
- OpenAI gym
- Roboschool


## Run

```
python main.py --env_name [name of environment]
```
